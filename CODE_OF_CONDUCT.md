# Code of Conduct

This project follows the **[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**.

We're committed to providing a welcoming and inclusive environment for everyone who participates in this project — contributors, users, and maintainers alike.

## Our pledge

Read the full Covenant at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>. The short version: be respectful, assume good faith, and engage constructively.

## Reporting

To report a violation of this Code of Conduct, contact the maintainers privately.

Use the same disclosure channel as security reports — see [SECURITY.md](SECURITY.md). Reports will be reviewed promptly and confidentially. We respect the privacy of everyone involved.

## Scope

This Code of Conduct applies in all project spaces — the GitHub repository, the issue tracker, pull requests, the project's chat/community channels (when they exist), and any other forum where someone is acting as a project participant.

## Enforcement

Maintainers may take any action they deem appropriate, including but not limited to: editing or removing offending content, temporarily or permanently banning a contributor from project spaces, or referring incidents to relevant external authorities when warranted.
